# EN
Hi there!
Here are implemented useful scripts for automating various processes that we often do manually and waste a lot of time on.

1) **/create_project_structure** - this is a script to convert your text project structure into real files and folders.
2) **/disable_windows_ services** - here are scripts that stop or remove unnecessary Windows 10-11 services to improve performance.
3) **/Save_and_open_folders** - here is a script that saves open at the Windows 11 directives-bookmarks in the active window in Explorer, and opens the saved directives if necessary. Useful if you have a lot of directives-bookmarks and need to quickly return to them in case of a PC reboot, etc.
4) **/stop_ports_process** - A collection of scripts to terminate processes listening on specific ports on Windows.
5) **/track_sys_process_realtime** - A Windows script that tracks the launch and termination of all processes in real time and displays where the process was launched from.

# UA
Привіт!
Тут реалізовано корисні скрипти для автоматизації різних процесів, які ми часто виконуємо вручну та витрачаємо на них багато часу.

1) **/create_project_structure** - тут скрипт для перетворення структури вашого текстового проекту на реальні файли та папки.
2) **/disable_windows_ services** - тут скрипти що зупиняють або видаляють непотрібні служби Windows 10-11 для покращення продуктивності.
3) **/Save_and_open_folders** - тут скрипт що зберігає відкриті в Windows 11 провіднику директиви-закладки з активного вікна, та при необхідності відкриває збережені директиви. Корисно якщо у вас багато директив-закладок і потрібно швидко повернутись до них в разі перезавантаження ПК, тощо.
4) **/stop_ports_process** – набір скриптів для завершення процесів, що прослуховують певні порти у Windows.
5) **/track_sys_process_realtime** - Windows-скрипт, який відстежує запуск та завершення всіх процесів у реальному часі та відображає звідки запущено процес.
