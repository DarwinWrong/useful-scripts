# AI CLI Control Panel

---

## EN

A convenient batch menu for managing AI CLI tools.

### Features

- **Run** installed AI CLI tools
- **Install** new programs via npm or pip
- **Update** existing installations
- **Uninstall** programs

### Supported Programs

| Program | Manager | Notes |
|---------|---------|-------|
| qwen-code | npm | AI coding assistant |
| GitHub Copilot | npm | GitHub's AI pair programmer |
| gemini-cli | npm | Google's Gemini CLI |
| codex | npm | OpenAI's Codex |
| claude-code | npm | Anthropic's Claude CLI |
| omniroute | npm | AI routing tool |
| openhands | pip | OpenHands AI agent |
| clawhub | npm | CLI hub manager |

### Usage

```batch
AI_CLI_Control_Panel.bat
```

Navigate the menu by entering the number of your choice and pressing Enter.

## UA

Зручне batch-меню для керування AI CLI інструментами.

### Функції

- **Запуск** встановлених AI CLI інструментів

AI_CLI_Control_Panel.bat

Переходьте по меню, вводячи номер обраного пункту та натискаючи Enter.
